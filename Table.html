Tabel HTML Dasar

Untuk membuat tabel di halaman web menggunakan HTML, Anda dapat menggunakan elemen < table > . Elemen < table > akan menampung seluruh elemen tabel lainnya, seperti baris < tr > dan sel < td >

Contoh Table:

Nama NPM Kelas Angkatan Semester Alamat

Rizal Falevi 2010631140113 A 2020 Semester 5 Karawang
